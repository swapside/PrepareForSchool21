Create a program that searches for the maximum number in a sequence of non-negative integers. The sequence is specified in the standard input streamâ€“stdin; the elements of the sequence are separated by spaces; the end of the sequence is -1, which marks the end of the sequence and is not one of its elements. The obtained maximum value must be output to the standard output stream. It is guaranteed that there is always at least one element in the sequence. There should be no newline break at the end of the response.

## Examples

| Input | Output |
| ------ | ------ |
| 1000 -1 | 1000 |
| 1 2 3 4 3 2 1 -1 | 4 |
